# Лабораторная работа №9. Адаптивная верстка: Медиа-запросы

## Цель работы

Изучить принципы адаптивной верстки и медиа-запросов. В конце
создать адаптивную страницу портфолио, которая корректно отображается на всех
устройствах

## Описание

1. Шаг 1. Создание рабочей директории
2. Шаг 2. Подключение рабочего каталога в VS Code
3. Шаг 3. Инициализация Git-репозитория
4. Шаг 4. Подключение к GitHub
5. Шаг 5. Создание базовой структуры
6. Шаг 6. Создание адаптивного макета
